# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 4.x     | Yes                |
| < 4.0   | No                 |

## Reporting a Vulnerability

If you discover a security issue in SwiftyChat, please report it responsibly:

1. **Do not** open a public GitHub issue.
2. Email **eneskaraosman53@gmail.com** with a description of the issue.
3. Include steps to reproduce, if possible.

You can expect an initial response within 72 hours. We will work with you to understand the scope and coordinate a fix before any public disclosure.

Thank you for helping keep SwiftyChat safe.
